A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup
Language, que significa Linguegem de Marcação de Hipertexto),
é uma linguagem de marcação utilizada na construção de páginas na
Web. Documentos HTML podem ser interpretados por navegadores

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tags associadas Descrição
Links <a> Criar links entre páginas
<ul> e <li> Criar listas de itens ordenados
<ol> e <il> Criar listas de itens ordenados
Imagens <limg> Inserir Imagens
